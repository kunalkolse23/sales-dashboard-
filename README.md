# 📊 Sales Dashboard Analysis (Excel Project)

This project involves the creation of an interactive sales dashboard using Microsoft Excel. The objective was to analyze raw sales data and transform it into meaningful insights that can support business decision-making. The dashboard provides a clear visualization of key performance metrics such as sales trends, customer distribution, and regional performance.

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning & Transformation
* Data Visualization (Charts & Graphs)

---

## 📊 Dashboard Features

The dashboard includes multiple analytical views to understand different aspects of sales performance:

* Orders vs Sales trend analysis to track monthly performance
* Gender-based sales distribution (Men vs Women)
* Age group segmentation (Young, Adult, Senior)
* Order status breakdown to identify fulfillment patterns
* Top 5 performing states based on revenue
* Sales contribution across different channels



---

## 📈 Key Insights

* A large percentage of total sales comes from a single order status (over 90%), indicating strong order completion performance
* Female customers contribute a higher share of sales compared to male customers
* A few key states significantly contribute to overall revenue, highlighting regional concentration
